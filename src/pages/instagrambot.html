Instagram Bot

CONFIGURATION

  • Username

    Instagram nickname without @

  • Password

    Your password is stored locally

  • Social algorithm

    Bot go to random hashtag from list, like 10-12 photo fast and stop X minutes in loop

  • Hashtags list

    10-15 hashtag is raccomanded. Separate with comma, example: murales, picoftheday, selfie

  • Max like for day

    Instagram limit is 1000/day, raccomanded value is 900

  • Numbers of like/user

    Number of likes to the same user

  • Max comments for day

    Number of comments for day, max is 600 raccomanded is 300

  • Comments list

    List of comments, example: wow, beautiful, amazing, nice photo

  • Max follow (and defollow) for day

    Max follow day, limit is 500 raccomanded is 300 (300 follow, 300 defollow)

  • Follows before rotate

    Follow 30 users, at 31 defollow first followed (in loop). Raccomanded is 30. If you set 0 the bot defollow all users (followed by bot previously)

  • Whitelist (not follow/defollow this user)

    Competitor user or enemies. NOTE: your following is excluded automatically

  • Competitor username

    Username, example (single user without @, not list): ptkdev

  • 2FA Pin (BETA)

    If you received sms or email pin, write it and wait

  • Google Chrome Path

    Install google chrome, this bot work with google chrome api

  • Debug Mode

    If bot don't work, enable this mode
